# project-candle
An application for student run organizations to promote, manage, and engage

The student org app will have 4 classes of users:
- **Admins** - These individuals will have complete authority over their student org and access to the financials; typically executive board members who can be      trusted to oversee the organization; Contributor with higher access privileges
- **Contributors** - These individuals will be able to create and promote events, start fundraisers, and update information about the org; typically board members who are responsible for running most aspects of the org; Admins with higher access privileges 
- **Members** - These individuals will be able to attend events, join discussion threads, and are general populace of the org; Followers with higher access privileges; May require membership fee (up to the org)
- **Followers** - These individuals will be able to view org pages and subscribe for updates; Lowest access privileges
